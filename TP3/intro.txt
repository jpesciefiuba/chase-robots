Funciones:

- Dibujado
    -> Dibujar Grilla
    -> Dibujar Robots
        -> Generar posición aleatoria de robots
    -> Dibujar Player
        -> Generar posición aleatoria de player
    -> Dibujar Colisiones
- Teletransportar jugador
    -> Averiguar y procesar ubicación del mouse
- Terminar Juego
    -> Detectar Teletransportar a Escombros
    -> Choque con robot
- Detectar Colision entre robots
- Mover
    -> Robots
    -> Jugador
    -> Escombros
- Pasar de nivel
    -> Generar posición aleatoria de player
    -> Generar posición aleatoria de robots
    -> aumentar la cantidad de robots
